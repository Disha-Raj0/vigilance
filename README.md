# Vigilance-X: AI Public Safety Intelligence

### Vision
To build a proactive defense against Digital Arrest scams and Organized Fraud.

### 🛠️ Core Features
- **Neural Link Analysis:** Connects victims to syndicates via shared UPI/IMEI data.
- **Psychological Scam Detection:** Gemini-powered intent analysis.
- **Forensic Vision:** AI-based counterfeit currency identification.
- **Command Center:** Real-time geospatial hotspot mapping.

### Tech Stack
- **AI:** Google Gemini 1.5 Flash/Pro
- **Intelligence logic:** NetworkX, Python
- **GIS:** Folium
- **Frontend:** Streamlit
